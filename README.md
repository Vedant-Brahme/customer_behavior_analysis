# 📊 Customer Shopping Behaviour Analysis

## 🔍 Overview

This project focuses on analyzing customer shopping behavior using a complete data analytics workflow. It includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and dashboard creation to generate actionable business insights.

The objective is to transform raw data into meaningful insights that can support data-driven decision-making.

---

## 📁 Dataset

* The dataset contains customer shopping details such as demographics, purchase behavior, and transaction-related information.
* Data includes structured fields suitable for analysis like customer ID, product categories, spending patterns, etc.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL (MySQL Server)**
* **Power BI**
* **Jupyter Notebook**

---

## ⚙️ Steps Performed

### 1. Data Loading

* Imported dataset into Python using Pandas
* Checked structure, data types, and initial records

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified trends in purchasing behavior
* Visualized key insights using charts and graphs

### 4. SQL Analysis

* Imported cleaned dataset into MySQL Server
* Performed queries to extract insights such as:

  * Top customers by spending
  * Most popular product categories
  * Sales trends

### 5. Dashboard Creation

* Built an interactive Power BI dashboard
* Included KPIs, filters, and visualizations for better understanding

---

## 📊 Dashboard

The Power BI dashboard provides:

* Key performance indicators (KPIs)
* Customer segmentation insights
* Sales and product performance analysis
* Interactive filters for dynamic exploration

---

## 📈 Results & Insights

* Identified high-value customers
* Discovered top-performing product categories
* Observed spending patterns across different segments
* Generated insights useful for business strategy and marketing decisions

---

## ▶️ How to Run

1. Clone or download the repository
2. Open the Jupyter Notebook file
3. Run all cells to perform data cleaning and EDA
4. Import the cleaned dataset into MySQL Server
5. Execute SQL queries for analysis
6. Open the Power BI file to explore the dashboard
